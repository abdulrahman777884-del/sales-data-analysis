## Project Overview
This project analyzes sales data to identify trends, customer behavior, profitability, and business performance across regions and sales channels.


## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
## Project Steps
1. Data Loading
2. Data Understanding
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Profit Analysis
6. Trend Analysis
7. Business Insights
8. Recommendations
## Key Insights
 The West region generated the highest sales.
 Some product categories achieved higher demand than others.
 Monthly sales showed seasonal fluctuations.
  Profitability varied across regions and sales channels.

## Visualizations
The project includes:
 Bar Charts
 Pie Charts
Line Charts
## Business Recommendations
 Focus on high-performing regions.
 Improve strategies during low-sales months.
 Optimize pricing and discount strategies.
 Improve customer experience through preferred payment methods.
